My HTML Cheat Sheet

Headings

Headings from `h1` to `h6` identify blocks and sections. Every page must have one `h1`.

Paragraphs

Paragraphs of text go in `p` tags.

Lists

Unordered Lists

Unordered lists are marked with bullet points and go in `ul` tags. Each list item goes in a `li` tag.

Types of lists:

  • Unordered lists (`ul`)
  • Ordered lists (`ol`)

Ordered Lists

Ordered lists are numbered and go in `ol` tags. Each list item goes in a `li` tag.

Polya's Problem Solving Framework

  1. Understand the problem
  2. Come up with a plan
  3. Carry out the plan
  4. Go back and improve your solution

Anchors (i.e. links)

Anchor tags create links to other pages. The URL goes in the `href` attribute. Go to DuckDuckGo to search for more information.

Setting `target="_blank"` will open the link in a new window. See MDN for more details.

Images

You can use the img tag to add images to your page.

Tables

Tables are used to display data in a neat spreadsheet format. The structure of a table is made up of a table head, body and footer. Data is placed in these locations using "row" or "column" scope.

Other content section tags

  • The 'article' HTML element represents a self-contained composition in a document, page, application, or site, which is intended to be independently distributable or reusable (e.g., in syndication). Examples include: a forum post, a magazine or newspaper article, or a blog entry, a product card, a user-submitted comment, an interactive widget or gadget, or any other independent item of content.
  • The 'footer' HTML element represents a footer for its nearest ancestor sectioning content or sectioning root element. A
    typically contains information about the author of the section, copyright data or links to related documents.
  • The 'header' HTML element represents introductory content, typically a group of introductory or navigational aids. It may contain some heading elements but also a logo, a search form, an author name, and other elements.
  • The 'nav' HTML element represents a section of a page whose purpose is to provide navigation links, either within the current document or to other documents. Common examples of navigation sections are menus, tables of contents, and indexes.
